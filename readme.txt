A simple yet powerful plugin that allows you to insert the Syrow Care code into your WordPress website.

== Description ==
This plugin allows you to easily integrate the Syrow Care code into your WordPress website.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/syrow-care` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Syrow Care screen to configure the plugin.

== Frequently Asked Questions ==
= How do I get the Syrow Care code? =
You can get the code from your Syrow Care account at care.syrow.com application.
